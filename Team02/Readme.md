# line 訊息整理師：提取聊天室的重點訊息
### Team members:姚奕慈
## Motivation
 2020推出的[LINE統計分析工具](https://chonyy.github.io/line-message-analyzer/)，輸入聊天記錄便可分析和朋友每日聊天訊息、常用詞，呈現統計圖表、文字雲。本次專題創新該工具，利用機器學習提取聊天記錄的重點，為LINE用戶標示群組對話重點，而後繪製文字雲、整合為摘要。
